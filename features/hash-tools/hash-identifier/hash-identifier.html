Supported Hash Types

MD5
Length: 32 hex chars
Output: 128 bits
Status: Deprecated

Legacy hash, vulnerable to collisions. Not recommended for security.

SHA-1
Length: 40 hex chars
Output: 160 bits
Status: Deprecated

Broken since 2017. Use SHA-256 or SHA-512 instead.

SHA-256
Length: 64 hex chars
Output: 256 bits
Status: Recommended

Industry standard for cryptographic hashing. Secure and widely used.

SHA-512
Length: 128 hex chars
Output: 512 bits
Status: Recommended

Strongest SHA-2 variant. Maximum security for critical applications.

bcrypt
Length: 60 chars
Format: $2a$, $2b$, $2y$
Status: Recommended

Adaptive password hashing. Includes salt and cost factor.

Argon2
Format: $argon2i$, $argon2d$
Winner: PHC 2015
Status: Best Choice

Most secure password hashing. Resistant to GPU/ASIC attacks.
