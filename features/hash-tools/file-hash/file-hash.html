Hash Algorithm Comparison

MD5
128-bit / 32 chars

Legacy algorithm. Fast but vulnerable to collisions. Not recommended for security.

SHA-1
160-bit / 40 chars

Deprecated since 2017. Broken by SHAttered attack. Use SHA-256 instead.

SHA-256
256-bit / 64 chars

Industry standard. Secure and widely used in blockchain and certificates.

SHA-512
512-bit / 128 chars

Maximum security. Best choice for critical applications.
