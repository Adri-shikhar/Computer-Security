Security Recommendations
  • Never reuse passwords across different accounts
  • Use a password manager to generate and store unique passwords
  • Enable 2FA (Two-Factor Authentication) wherever possible
  • Use passphrases (4-5 random words) for memorable but secure passwords
  • Change breached passwords immediately on all sites where used

How It Works

1. Local Hashing

Your password is hashed with SHA-1 in your browser

2. Prefix Query

Only first 5 characters sent to HIBP API

3. Local Verification

Results matched locally in your browser
