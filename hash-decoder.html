Intelligent Hash Analyzer

Automatically detects algorithm type and extracts parameters
Smart Detection: Paste any hash and we'll automatically identify its format (MD5, SHA-1, BCrypt, Argon2, PBKDF2) and extract embedded parameters like salt, iterations, and memory cost.
Supports: MD5 (32 hex), SHA-1 (40 hex), BCrypt ($2a$/$ 2b$), Argon2 ($argon2*$), PBKDF2 (pbkdf2:*)
Security Note: Hash format reveals algorithm details but doesn't compromise security if strong algorithms are used
